# Ram Pravesh Singh Portfolio

Personal portfolio built with Next.js to showcase frontend engineering work across React.js, Next.js, React Native, and TypeScript.

## Overview

This portfolio highlights:

- Resume-aligned frontend engineering experience
- Web and mobile product case studies
- GitHub and open-source contribution proof
- Recruiter-friendly project links and downloadable resume access

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Sections

- Hero
- Selected Work
- Credibility Strip
- Experience Snapshot
- Technical Depth
- GitHub / Open Source
- Working Style
- Contact CTA

## Featured Projects

- Frontend Audit Extension
  React, TypeScript, Vite, Chrome Manifest V3, Tailwind CSS, Chrome Storage API
- Smart Ecommerce Platform & Admin Dashboard
  Next.js, TypeScript, Tailwind CSS, TanStack Query, Zustand, React Hook Form, Zod, MSW, Vercel
- Smart Service Booking App
  React Native, Expo, TypeScript, TanStack Query, Zustand, React Hook Form, Zod, EAS Build, Google Play Console

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```

## Deployment

The portfolio is structured for Vercel deployment and uses App Router metadata plus Open Graph image generation.
